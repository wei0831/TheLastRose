# Prototypes Content

* * *

* **w_MouseClick:** Added 10/06/2013
	* Mouse click to dispatch event. Objects that has reciever and if it is within range will respond to the event.

* **w_SoulAbsorb_Part1:** Added 10/08/2013
	* Press Key.Space to spawn souls. Demonstrate simple soul absorb using particle effects.

* **w_SoulAbsorb_Part2:** Added 10/09/2013
	* Press Key.Space to spawn ability souls. Demonstrate soul raise from dead monster. It can be broken into many souls.

* **w_CameraPrototype:** Added 10/13/2013
	* Press Key.Q: Camera Shake
	* Press Key.W: Camera FadeIn
	* Press Key.E: Camera FadeOut
	* Press Key.R: Toggle Camera Follow
	* Press Key.T: Toggle Camera Chase
	* Mose Click: Shift Camera
	* Camera Shake
		* Shaking in X with radius, Shaking in Y with radius
		* Shaking with timer(>=0) with delay(>= 0) in between
	* Camera Fade
		* Start Color, End Color, Change Rate(0.0 ~ 1.0), delay(>=0) in between
		
* **w_LightTest:** Added 10/16/2013
	* Naive Lighting simulation.

* **w_RespawnTest:** Added 10/27/2013
	* Die -> Respawn at CheckPoint

* **w_Weather:** Added 10/28/2013
	* Raining
	* Press Key.Space to Activate Thunder

* **w_PauseTest:** Added 11/1/2013
	* Press Key.Space: Pause
	
* **w_LightTest3:** Added 11/1/2013

* **w_IntroTitle:** Added 11/11/2013
